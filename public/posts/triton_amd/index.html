Optimizer

Optimization Passes:

  1. MLIR general optimization passes, like CSE, DCE, Inlining, etc.
  2. GPU specific optimization passes, like Pipeline, Prefetch, Matmul accelerate, Coalesce, etc.
  3. Vender specific GPU optimization passes, for example Nvidia provides TMA, Async Dot, etc. AMD provides OptimizeLDSUsage, BlockPingpong, etc.

ttir (Triton IR)

make_ttir_funtion() hardware independent Passes: inline optimization/CSE/Canonicalization/DCE/Loop Invariant Code Motion/Loop Unroll

ttgir (Triton GPU IR)

make_ttgir() hardware dependent designed for GPU platform to boost its performance Passes: AMD GPU Matmul/Block PingPong/Pipeline etc

llir (LLVM-IR)

make_llir() IR-level optimization and AMD GPU specific optimization
